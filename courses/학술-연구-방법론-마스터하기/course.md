# 연구 주제 선정과 문헌 리뷰

좋은 연구는 좋은 질문에서 시작된다

---

# 좋은 연구 주제의 조건

- 구체적이고 측정 가능한 연구 질문
- 기존 연구와의 차별점 (연구 갭)
- 실현 가능한 범위와 자원
- 학문적/실무적 기여도

---

# 문헌 검색 도구

- Google Scholar: 가장 범용적인 학술 검색 엔진
- PubMed: 의생명과학 분야 논문 DB
- RISS: 국내 학위논문·학술지 통합 검색
- Web of Science / Scopus: 고품질 학술지 색인
- Zotero / Mendeley: 문헌 관리 도구

# 연구 방법론

---

# 정량 연구

- 숫자로 측정 가능한 데이터 수집
- 설문조사, 실험, 기존 데이터 분석
- 가설 검증에 적합 (연역적 접근)
- 통계 분석으로 일반화 가능

---

# 정성 연구

- 텍스트, 이미지, 관찰 등 비수치 데이터
- 심층 인터뷰, 사례 연구, 민족지학
- 현상의 맥락과 의미 파악 (귀납적 접근)
- 풍부한 서술로 깊이 있는 이해 제공

# 통계 분석 기초

데이터로 가설을 검증하기

---

# 주요 통계 분석 방법

- t-test: 두 그룹 간 평균 비교
- ANOVA: 세 그룹 이상의 평균 비교
- 상관분석: 두 변수 간의 관계 강도
- 회귀분석: 변수 간 인과관계 추정
- 카이제곱 검정: 범주형 변수 간 독립성 검정

---

# p-value 이해하기

- p < 0.05: 통계적으로 유의미 (귀무가설 기각)
- p-value는 '효과의 크기'가 아닌 '우연의 확률'
- 효과 크기(Effect Size)도 함께 보고해야 함
- 유의성만으로 실질적 의미를 판단하면 안 됨

# 논문 작성과 투고

연구 결과를 세상에 알리기

---

# 논문의 기본 구조 (IMRaD)

- Introduction: 연구 배경, 목적, 가설
- Methods: 연구 대상, 도구, 절차
- Results: 분석 결과 (표, 그래프)
- Discussion: 결과 해석, 시사점, 한계점
- Abstract: 전체를 요약한 초록 (마지막에 작성)

---

# 투고 과정

- 1. 타겟 저널 선정 (IF, 분야 적합성)
- 2. 저널 투고 규정에 맞게 포맷팅
- 3. 온라인 시스템으로 원고 제출
- 4. 동료 심사(Peer Review) 대응
- 5. 수정(Revision) 후 재투고 → 게재 승인
