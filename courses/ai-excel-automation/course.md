---

# 엑셀 수식, 왜 어려운가

- VLOOKUP, INDEX/MATCH, SUMIFS... 함수 이름부터 외계어
- 중첩 함수(함수 안에 함수)가 들어가면 해독 불가
- 구글링해도 내 상황에 딱 맞는 답을 찾기 어려움
- AI에게 한국어로 "이거 해줘"라고 하면 수식을 만들어줍니다

![엑셀 자동화 흐름](https://www.vibecamp.us/course-images/excel-flow.svg)

---

# 기본 수식 요청 프롬프트

- "A열에 이름, B열에 부서, C열에 매출이 있어. 영업부의 매출 합계를 구하는 수식을 알려줘"
- AI 답변: =SUMIF(B:B,"영업부",C:C)
- "이 수식이 어떻게 작동하는지 초보자도 이해하게 설명해줘"
- 수식 + 설명을 같이 받으면 다음에 직접 응용 가능

---

# VLOOKUP을 자연어로

- 상황: "거래처 코드로 거래처명을 자동으로 가져오고 싶어"
- AI에게: "A열에 거래처 코드가 있고, 별도 시트의 A열에 코드, B열에 거래처명이 있어. A열 코드로 거래처명을 가져오는 수식 만들어줘."
- AI 답변: =VLOOKUP(A2,거래처목록!A:B,2,FALSE)
- 더 나아가: "VLOOKUP 대신 더 좋은 방법이 있어?" → XLOOKUP 추천

---

# 조건부 수식 요청하기

- "매출이 1000만원 이상이면 '달성', 미만이면 '미달성'으로 표시"
- AI 답변: =IF(C2>=10000000,"달성","미달성")
- "조건이 3개야: 1000만 이상 '우수', 500만 이상 '보통', 미만 '미달'"
- AI 답변: =IF(C2>=10000000,"우수",IF(C2>=5000000,"보통","미달"))
- 복잡한 중첩 IF도 자연어 한 줄이면 끝

---

# 실습: 내 업무에 맞는 수식 받기

- 평소 엑셀에서 손으로 하던 반복 작업을 떠올리기
- "매달 수동으로 하는 이 계산을 수식으로 바꿔줘"라고 요청
- AI가 준 수식을 엑셀에 직접 입력해서 테스트
- 안 되면 에러 메시지를 AI에게 보여주고 수정 요청

---

<!-- +qa -->
- q: "AI가 준 수식이 에러가 나요"
  a: "에러 메시지(예: #N/A, #VALUE!, #REF!)를 그대로 AI에게 알려주세요. '이 수식에서 #N/A 에러가 나는데 왜 그런지, 어떻게 고치는지 알려줘'라고 하면 원인과 해결책을 알려줍니다. 데이터 범위나 시트 이름이 다른 경우가 가장 흔한 원인입니다."

  a: "AI에게 '엑셀 2016 버전이야'라고 알려주세요. XLOOKUP 같은 최신 함수 대신 VLOOKUP이나 INDEX/MATCH로 대체 수식을 만들어줍니다. Google Sheets용 수식도 따로 요청할 수 있습니다."

# 복잡한 수식 해독과 응용

전임자가 남긴 괴물 수식, AI로 해독하기

---

# 기존 수식 해석 요청

- 전임자가 만든 수식이 뭔지 모르겠을 때
- "이 수식을 한국어로 설명해줘: =INDEX(B:B,MATCH(1,(A:A=F1)*(C:C="완료"),0))"
- AI가 단계별로 분해해서 설명해줌
- "이 수식을 더 간단하게 바꿀 수 있어?" → 최적화 제안

---

# 날짜/텍스트 함수 요청

- "주민등록번호 앞자리에서 생년월일을 추출해줘"
- "날짜에서 요일만 한국어로 표시해줘 (월, 화, 수...)"
- "이름과 부서를 합쳐서 '홍길동(영업팀)' 형태로 만들어줘"
- AI는 TEXT, LEFT, RIGHT, MID, CONCATENATE 등을 조합해서 알려줌

---

# 배열 수식과 고급 함수

- "여러 조건을 동시에 만족하는 데이터를 찾고 싶어"
- SUMPRODUCT, COUNTIFS, AVERAGEIFS 같은 다중 조건 함수
- AI에게 조건을 나열하면 적절한 함수를 선택해서 수식 작성
- "Ctrl+Shift+Enter 배열 수식이 필요한지도 알려줘"

---

# 수식 디버깅 요청

- "이 수식이 일부 행에서만 작동하고 나머지는 에러가 나"
- "원인을 찾아줘: =VLOOKUP(A2,Sheet2!A:C,3,FALSE) → #N/A"
- AI 진단: "Sheet2의 A열에 앞뒤 공백이 있을 수 있어요. TRIM 함수를 추가하세요."
- AI 수정: =VLOOKUP(TRIM(A2),Sheet2!A:C,3,FALSE)

---

# 실습: 괴물 수식 해독하기

- 회사 엑셀 파일에서 가장 복잡한 수식을 찾아서 AI에게 질문
- "이 수식을 단계별로 설명해줘"
- "이 수식을 더 효율적으로 바꿀 수 있어?"
- "이 수식이 하는 일을 다른 방법으로도 할 수 있어?"

---

# 조건부 서식과 데이터 유효성

보기 좋은 엑셀이 일도 잘하는 엑셀입니다

---

# 조건부 서식 요청

- "매출 목표 달성한 셀은 초록색, 미달성은 빨간색으로 표시해줘"
- "마감일이 지난 항목은 빨간 글자로 표시하는 조건부 서식 알려줘"
- "상위 10% 데이터를 자동으로 강조 표시하고 싶어"
- AI가 조건부 서식 규칙과 적용 방법을 단계별로 안내

---

# 데이터 유효성 검사 설정

- "부서 선택을 드롭다운 목록으로 만들어줘"
- "날짜 열에 날짜 형식만 입력되게 제한하고 싶어"
- "금액 열에 음수가 입력되지 않게 해줘"
- 데이터 입력 오류를 사전에 방지하는 강력한 기능

---

# 실용적인 서식 템플릿

- 신호등 서식: 진행 상태를 빨강/노랑/초록으로 자동 표시
- 히트맵: 숫자 크기에 따라 색상 농도 변경
- 데이터 바: 셀 안에 막대그래프 표시
- AI에게 "이런 효과를 내고 싶어"라고 설명하면 설정 방법을 알려줌

---

# 실습: 내 엑셀 파일 꾸미기

- 자주 쓰는 엑셀 파일에 조건부 서식 3개 이상 적용
- 드롭다운 목록 1개 이상 만들기
- AI에게 "이 엑셀 시트를 보고하기 좋게 꾸미는 방법 알려줘" 요청
- 시각적으로 정리된 엑셀은 업무 효율도 높여줍니다

# 흔한 데이터 문제들

- 중복 데이터: 같은 거래가 2번 입력됨
- 공백 문제: 셀 앞뒤에 보이지 않는 공백
- 형식 불일치: "2024-01-15", "2024/01/15", "1월 15일" 혼재
- 빈 셀: 데이터가 누락된 행
- AI에게 "이 데이터를 정리해줘"라고 하면 단계별로 안내

# 중복 제거 프롬프트

- "A열 기준으로 중복 데이터를 찾는 수식 알려줘"
- AI: =COUNTIF(A:A,A2)>1 → TRUE면 중복
- "중복을 찾아서 색칠하는 조건부 서식도 알려줘"
- "중복 중 최신 데이터만 남기고 나머지는 삭제하는 방법은?"

# 텍스트 데이터 정리

- "이름에 공백이 섞여 있어. 정리하는 수식 알려줘" → TRIM
- "전화번호를 010-1234-5678 형식으로 통일하고 싶어"
- "주소에서 시/도만 추출하고 싶어"
- AI에게 현재 형식과 원하는 형식을 알려주면 변환 수식 제공

---

# 날짜/숫자 형식 통일

- "텍스트로 저장된 날짜를 실제 날짜로 변환해줘"

- "단위가 만원인 것과 원인 것을 통일하고 싶어"

# 피벗 테이블 만들기

수만 행 데이터를 한눈에 요약하는 마법

---

# 피벗 테이블이란?

- 대량의 데이터를 원하는 기준으로 집계하는 기능
- 부서별 매출 합계, 월별 주문 건수, 제품별 평균 단가 등
- 수식 없이 드래그 앤 드롭으로 만들 수 있음
- AI에게 "어떤 피벗 테이블을 만들면 좋을지" 추천받기

---

# AI에게 피벗 테이블 설계 받기

- "이 데이터에 날짜, 부서, 제품, 매출 열이 있어. 유용한 피벗 테이블을 추천해줘."
- AI 추천: 1) 월별 총 매출, 2) 부서별/제품별 매출 교차표, 3) 분기별 성장률
- "피벗 테이블 만드는 방법을 엑셀 버전에 맞게 단계별로 알려줘"
- 초보자도 따라할 수 있는 클릭 단위 가이드 제공

---

# 피벗 테이블 활용 패턴

- 월간 매출 보고: 날짜를 월 단위로 그룹화, 부서별 소계
- 거래처 분석: 거래처별 주문 건수, 평균 금액, 총액
- 재고 현황: 제품별 입고/출고/잔고 집계
- AI에게 "우리 팀 업무에 맞는 피벗 테이블 아이디어를 줘" 요청

---

# 피벗 테이블 트러블슈팅

- "피벗 테이블이 새 데이터를 반영 안 해요" → 데이터 원본 범위 확인, 새로고침
- "날짜가 그룹화되지 않아요" → 텍스트로 저장된 날짜 문제
- "빈 행이 나와요" → 원본 데이터에 빈 셀 확인
- 문제 상황을 AI에게 설명하면 원인과 해결책을 안내

---

# 실습: 첫 피벗 테이블 만들기

# 차트와 시각화

숫자를 그림으로 바꾸면 설득력이 10배 올라갑니다

---

# 어떤 차트를 써야 할까?

- 추이/변화: 꺾은선 그래프 (월별 매출 추이)
- 비교: 막대 그래프 (부서별 실적 비교)
- 비율/구성: 원형 그래프 (비용 구성 비율)
- 상관관계: 분산형 차트 (광고비 vs 매출)
- AI에게 "이 데이터에 맞는 차트 유형을 추천해줘"

---

# AI로 차트 설계하기

- "월별 매출 데이터가 있어. 보고서에 넣을 차트를 추천해줘."
- "이 차트에 목표선을 추가하고 싶어. 방법 알려줘."
- "차트 제목, 축 레이블, 범례를 전문적으로 설정하는 법 알려줘."
- AI가 차트 종류 선택부터 세부 설정까지 가이드

---

# 보고서용 차트 만들기 팁

- 색상: 회사 브랜드 컬러 사용, 3색 이내로 제한
- 레이블: 핵심 수치만 표시, 소수점 통일
- 제목: "매출 현황"(X) → "4분기 매출 전년 대비 15% 성장"(O)
- 불필요한 장식 제거: 3D 효과, 그라데이션은 오히려 방해

---

# 차트 + 인사이트 결합

- 차트만 던지면 "그래서 뭐?"라는 반응
- AI에게 "이 차트 데이터를 보고 인사이트를 3개 뽑아줘"
- "이 추이가 계속되면 연말에 어떤 결과가 예상되는지 분석해줘"
- 차트 아래에 한 줄 해석을 넣으면 보고서 품질이 올라갑니다

---

# 실습: 업무 데이터 시각화

- 업무 데이터로 차트 2~3개 만들기
- AI에게 차트 개선 포인트 요청
- "이 차트를 프레젠테이션에 넣을 수 있게 다듬어줘" 요청
- 최종 차트를 보고서에 삽입하여 활용

# VBA 코드를 AI에게 받기

코딩 몰라도 됩니다. AI가 대신 짜줍니다.

---

# VBA가 뭔가요?

- Visual Basic for Applications: 엑셀에 내장된 프로그래밍 언어
- 반복 작업을 버튼 하나로 자동 실행
- 예: 매월 같은 형식의 보고서 생성, 데이터 정리, 이메일 발송
- 코딩을 몰라도 AI가 코드를 만들어주니 걱정 없음

![엑셀 자동화 흐름](https://www.vibecamp.us/course-images/excel-flow.svg)

---

# VBA 요청 프롬프트

- "엑셀 VBA로 A시트의 데이터를 부서별로 분리해서 각각 새 시트로 만드는 코드를 작성해줘"
- "매월 1일에 전월 데이터를 백업 시트로 이동하는 VBA 코드 만들어줘"
- "선택한 범위의 빈 셀을 위쪽 값으로 채우는 매크로 만들어줘"
- AI에게 "하고 싶은 일"만 설명하면 코드가 나옵니다

---

# VBA 코드 적용하는 법

- 1단계: Alt + F11 → VBA 편집기 열기
- 2단계: 삽입 → 모듈 → 새 모듈 만들기
- 3단계: AI가 준 코드를 붙여넣기
- 4단계: F5 또는 실행 버튼으로 실행
- "이 과정을 스크린샷과 함께 설명해줘"라고 AI에 요청 가능

---

# 안전하게 VBA 사용하기

- 반드시 파일을 백업한 후 실행
- AI 코드를 실행 전에 "이 코드가 정확히 뭘 하는지 설명해줘" 확인
- 처음에는 작은 데이터로 테스트
- 문제 발생 시 Ctrl+Z(실행 취소)가 안 되는 경우도 있으니 백업 필수

---

# 실습: 첫 VBA 매크로 만들기

- 일상 업무 중 반복되는 작업 1가지 선택
- AI에게 "이 작업을 VBA로 자동화해줘" 요청
- 코드를 받아서 테스트 파일에 적용
- 작동하면 실제 업무에 활용, 안 되면 에러를 AI에게 전달

---

<!-- +qa -->
- q: "VBA 코드를 실행했는데 보안 경고가 나와요"
  a: "엑셀 보안 설정 때문입니다. 파일 → 옵션 → 보안 센터 → 매크로 설정에서 '알림을 표시하고 사용자가 선택'으로 변경하세요. 회사 IT 정책에 따라 매크로가 차단될 수 있으니 IT팀에 확인도 필요합니다."

  a: "AI에게 코드를 붙여넣고 '이 코드에서 [이 부분]을 [이렇게] 바꾸고 싶어'라고 하면 됩니다. 코딩 지식이 전혀 없어도 자연어로 수정 요청할 수 있습니다."

# Google Apps Script 활용

구글 시트 사용자를 위한 무료 자동화 도구

---

# Apps Script란?

- 구글 시트에 내장된 자동화 도구 (JavaScript 기반)
- 구글 계정만 있으면 무료로 사용 가능
- 구글 시트, 문서, 캘린더, Gmail과 연동 가능
- 예: 시트 데이터를 자동으로 이메일로 발송

---

# Apps Script 요청 프롬프트

- "구글 시트에서 A열이 '완료'인 행을 자동으로 '완료' 시트로 이동하는 스크립트 만들어줘"
- "매일 오전 9시에 시트의 데이터를 요약해서 나에게 이메일 보내는 스크립트"
- "구글 폼 응답이 들어오면 자동으로 시트를 업데이트하는 스크립트"

---

# Apps Script 적용하는 법

- 1단계: 구글 시트 → 확장 프로그램 → Apps Script
- 2단계: 편집기에 AI가 준 코드 붙여넣기
- 3단계: 저장 → 실행 → 권한 승인
- 4단계: 트리거 설정 (매일/매시간 자동 실행)
- AI에게 "트리거 설정하는 방법도 알려줘" 요청

---

# 실용적인 Apps Script 예시

- 월말 자동 보고서: 데이터를 집계해서 요약 시트 생성
- 마감일 알림: 마감 3일 전 자동 이메일 발송
- 데이터 검증: 입력된 데이터가 규칙에 맞는지 자동 체크
- 대시보드 자동 갱신: 원본 데이터가 바뀌면 대시보드 자동 업데이트

---

# 실습: 자동 이메일 알림 만들기

- 구글 시트에 마감일이 있는 업무 목록 만들기
- AI에게 "마감일 3일 전에 이메일 알림 보내는 스크립트 만들어줘" 요청
- 코드 적용 후 트리거 설정
- 자동 알림이 오는지 테스트

---

# 업무 자동화 설계

반복 업무를 찾아내고, 자동화 계획을 세우기

---

# 자동화할 업무 찾기

- 매일/매주/매월 반복하는 작업 목록 작성
- "복사-붙여넣기"를 많이 하는 작업 = 자동화 후보
- 같은 형식의 파일을 반복 생성하는 작업
- AI에게 "내가 하는 이런 업무를 자동화할 수 있을까?" 질문

---

# 자동화 난이도 판단

- 쉬움: 수식으로 해결 (VLOOKUP, IF, SUMIF)
- 보통: 피벗 테이블 + 조건부 서식
- 약간 어려움: VBA/Apps Script (AI가 코드 작성)
- 고급: 외부 API 연동, 여러 시스템 통합
- AI에게 "이 작업의 자동화 난이도와 방법을 알려줘" 요청

---

# 자동화 프로젝트 계획

- 1주차: 반복 업무 3개 선정, AI에게 자동화 방법 상담
- 2주차: 가장 쉬운 것 1개 자동화 (수식 또는 조건부 서식)
- 3주차: VBA 또는 Apps Script로 1개 자동화
- 4주차: 결과 평가, 다음 자동화 대상 선정
- 작은 성공부터 시작하는 것이 핵심

---

# 자동화 전후 비교

- "이 작업에 기존에 몇 분 걸렸고, 자동화 후 몇 분인지 비교해줘"
- 시간 절감 효과를 수치로 보여주면 상사도 납득
- 예: 주간보고용 데이터 정리 30분 → 5분 (매주 25분 절약, 연간 20시간)
- AI에게 "이 자동화의 ROI를 계산해줘" 요청 가능

---

# 지속 가능한 자동화 관리

- 자동화 스크립트에 주석(설명) 달아두기 → AI에게 "이 코드에 한국어 주석 달아줘"
- 담당자가 바뀌어도 유지보수 가능하도록 문서화
- 데이터 형식이 바뀌면 수식/코드도 업데이트 필요
- "이 자동화가 깨지는 상황은 뭐가 있을까?" AI에게 미리 확인

---

<!-- +qa -->
- q: "자동화했는데 오히려 일이 더 많아진 것 같아요"
  a: "처음 세팅할 때는 시간이 걸립니다. 하지만 한 번 만들어두면 매번 반복할 때마다 시간을 아끼게 됩니다. 3번 이상 반복할 작업만 자동화하세요. 1~2번만 할 작업은 수동이 더 빠릅니다."
- q: "코딩을 전혀 모르는데 자동화를 할 수 있나요?"
  a: "네, 이 과정의 핵심이 바로 그겁니다. 코딩은 AI가 하고, 여러분은 '뭘 자동화할지'만 결정하면 됩니다. '이런 작업을 자동으로 하고 싶어'라고 자연어로 설명하는 능력이 코딩 실력보다 훨씬 중요합니다."
